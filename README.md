# 机器学习课程实验仓库

本仓库用于整理机器学习课程的实验代码、数据、报告和结果文件，便于复现、提交和回顾。

## 仓库概览

当前仓库包含 4 个实验，分别对应常见的分类与建模任务：

- `实验一 贝叶斯分类器设计与实现/`：中文外卖评论情感分类。
- `实验二 决策树算法的设计与实现/`：鸢尾花数据集上的手写决策树。
- `实验三 人工神经网络的设计与实现/`：基于鸢尾花数据集的感知机实验。
- `实验四 垃圾邮件分类器文本分类设计与实现/`：英文短信垃圾邮件分类。

## 目录结构

通常每个实验目录都包含以下内容：

- `src/`：实验主程序。
- `data/`：数据集、词表或其他输入文件。
- `docs/`：实验指导书、分析说明、报告文档。
- `results/`：运行后生成的图表、CSV 和日志等输出文件。
- `README.md`：该实验的独立说明文档。

## 快速开始

1. 进入对应实验目录。
2. 安装该实验所需依赖。
3. 运行 `src/` 下的主程序。

示例：

```powershell
cd ".\实验一 贝叶斯分类器设计与实现"
python .\src\bayes_classifier_experiment.py
```

```powershell
cd ".\实验二 决策树算法的设计与实现"
python .\src\decision_tree_experiment.py
```

```powershell
cd ".\实验三 人工神经网络的设计与实现"
python .\src\perceptron_experiment.py
```

```powershell
cd ".\实验四 垃圾邮件分类器文本分类设计与实现"
python .\src\spam_classifier_experiment.py
```

## 依赖说明

各实验的依赖文件放在对应目录中，通常是：

- `requirements.txt`
- `requirement.txt`

安装前请先查看实验目录内的 README，确认数据路径、参数和运行方式。

## 输出结果

各实验运行后通常会在 `results/` 目录生成以下类型文件：

- 结果对比表：如 `prediction_compare.csv`
- 评价指标表：如 `metrics.csv`
- 可视化图像：如分布图、混淆矩阵、分类树图

具体输出以各实验目录内的说明为准。

## 课程说明

本仓库以课程实验整理为主，重点是保留可复现的代码、数据和实验结论。

如果你需要提交课程作业，建议优先检查：

- 数据文件是否完整。
- 代码路径是否与当前目录结构一致。
- 运行结果是否与本机环境一致。

## AI 参与说明

本仓库中的部分代码整理、文档润色和说明文字可能经过 AI 辅助完成。

如用于课程提交，请自行确认：

- 代码逻辑是否符合实验要求。
- 结果是否与实际运行一致。
- 实验结论是否经过人工检查。

## 维护建议

- 新增实验时，继续沿用“实验目录 + `src`/`data`/`docs`/`results`”的结构。
- 每个实验最好单独维护一份 README，写清楚输入、运行方式和输出。
- 如果后续补充实验报告，建议把最终版文档放入 `docs/`。

## Commit 规范

建议使用简洁、统一的提交信息格式：

```text
<type>: <description>
```

常用 `type`：

- `feat`：新增实验内容、功能或代码实现。
- `fix`：修复错误或异常。
- `docs`：只修改文档。
- `refactor`：重构代码，但不改变主要功能。
- `test`：新增或调整测试。
- `chore`：杂项维护、配置整理或依赖调整。

示例：

- `feat: 完成贝叶斯分类器实验`
- `docs: 更新README说明`
- `fix: 修正决策树输出结果`
