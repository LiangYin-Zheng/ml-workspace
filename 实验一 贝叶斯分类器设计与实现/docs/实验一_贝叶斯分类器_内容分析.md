## 4. 实现流程

### 4.1 数据读取与检查

程序读取 CSV 后，先输出前 5 行数据，然后检查 `review` 和 `label` 的缺失情况以及重复样本数量。之后会删除重复数据和缺失数据，保证后续建模输入有效。

### 4.2 文本清洗

清洗函数 `data_cleaning(content_list, stopwords)` 会对每条评论执行以下操作：

1. 将文本统一转换为字符串。
2. 去除特殊字符、标点、数字和英文字母。
3. 使用 `jieba.cut(content, cut_all=False)` 分词。
4. 去除停用词、换行符、全角空格和普通空格。
5. 用空格连接分词结果，写入 `review_clean` 列。

### 4.3 向量化与建模

程序使用 `TfidfVectorizer(max_features=3000)` 对清洗后的文本做向量化，再将稀疏矩阵转换为数组，以满足 `GaussianNB` 的输入要求。之后使用 `train_test_split()` 划分训练集和测试集，并在测试集上计算评价指标。

### 4.4 结果保存

程序会保存两类结果：

- `label_distribution.png`：用于展示类别分布。
- `prediction_compare.csv`：用于对比测试集评论的真实值和预测值。

1. 请先将 `takeout_reviews.csv` 和 `hit_stopwords.txt` 放到 `data/` 目录，或者使用绝对路径参数。
2. `GaussianNB` 需要稠密矩阵，因此代码中将 TF-IDF 结果转为数组是必要步骤。
3. 如果当前 Python 环境缺少 `jieba`、`pandas`、`numpy`、`matplotlib`、`seaborn` 或 `scikit-learn`，需要先准备好对应依赖后再运行。
