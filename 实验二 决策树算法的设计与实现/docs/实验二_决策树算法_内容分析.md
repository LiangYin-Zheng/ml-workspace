## 4. 实现流程

### 4.1 数据读取与探索

程序先加载鸢尾花数据集，并输出前 5 行、数据集形状、缺失值统计、类别分布和类别名称。

### 4.2 基尼系数计算

函数 `gini(y)` 先统计当前节点中各类别样本数量，再按公式计算节点不纯度：

```text
Gini(t) = 1 - sum(p_c ** 2)
```

其中 `p_c` 表示第 `c` 类样本在当前节点中的比例。

### 4.3 决策树生成

函数 `generate()` 是递归生成决策树的核心。它会先记录当前节点的类别分布，然后判断是否满足停止条件。若不停止，则遍历所有特征与候选阈值，按 `X[f] <= v` 和 `X[f] > v` 划分样本，并选择不纯度下降量最大的切分点。

### 4.4 模型预测

预测时从根节点开始，根据当前节点的切分特征和切分阈值向左或向右移动，直到到达叶子节点。叶子节点中样本数量最多的类别即为预测类别。

### 4.5 可视化与结果保存

程序使用 `networkx.DiGraph` 表示树结构，再通过 `nx.draw_networkx()` 绘制并保存为 `decision_tree_visualization.png`。同时会导出预测对比表和指标文件，便于报告编写与结果核对。

## 7. 注意事项

1. 当前程序没有直接调用 `sklearn.tree.DecisionTreeClassifier`，而是手动实现决策树生成逻辑。
2. `sklearn` 仅用于加载数据、划分训练测试集和计算评价指标。
3. 决策树的深度、叶子节点最少样本数与最终结果强相关，修改参数后建议重新核对输出。
